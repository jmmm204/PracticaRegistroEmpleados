Este proyecto es una aplicación de consola en Java diseñada para gestionar el registro de empleados y la asignación de cargos, utilizando JPA con Hibernate para el mapeo objeto-relacional (ORM), lo que permite una comunicación fluida con la base de datos. El núcleo del sistema se compone de las entidades Cargo y Empleado, que representan las tablas de la base de datos y están vinculadas por una relación @ManyToOne. La lógica de persistencia se abstrae en una capa de repositorio (patrón DAO), donde una interfaz genérica ICRUD define las operaciones y las clases DCargo y DEmpleado las implementan, utilizando un EntityManager gestionado por la clase JPAUtil y configurado mediante el archivo persistence.xml. Finalmente, la clase Main actúa como punto de entrada, orquestando la creación, lectura, actualización y eliminación de registros para demostrar la funcionalidad completa del sistema.
